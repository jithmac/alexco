# Permissions Audit


## Admin
- **admin.manage**: Full Admin Access
- **admin.view**: View Admin Dashboard

## E-commerce
- **ecommerce.manage**: Manage Online Orders

## HR
- **hr.manage**: Manage Employees & Payroll
- **hr.view**: View HR Dashboard

## Inventory
- **inventory.manage**: Manage Inventory (Add/Edit/Audits)
- **inventory.view**: View Inventory

## POS
- **pos.access**: Access POS Terminal

## System
- **users.manage**: Manage Users & Roles

## Tickets
- **tickets.manage**: Manage Job Tickets
